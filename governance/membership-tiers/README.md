# Membership Tiers

The **Membership Tier** describes the level of engagement, contribution and responsibilities of a member which helps define their privileges within the community.

{% hint style="info" %}
Keep in mind that we operate in a culture of mutual respect. This membership framework should in no way define or influence how we respect and treat each other.
{% endhint %}



{% stepper %}
{% step %}
### Explorer

Guests who want to learn about The Community Workshop, and explore what they can build, contribute and learn.
{% endstep %}

{% step %}
### Tinkerer

The entry-level membership. Members who want to build personal and community projects. Maybe new to skill or technical knowledge. Might still be exploring what they want to build or focus on.
{% endstep %}

{% step %}
### Innovator

Members who have been consistently working on personal and contributing contributing to community projects.
{% endstep %}

{% step %}
### Trusted Innovator

Members who have been consistently working on personal and contributing to community projects.
{% endstep %}

{% step %}
### Keystone Innovator

Highest trust rating for members, only select few can be Keystone Innovators.
{% endstep %}

{% step %}
### Governance

Highest trust rating for members, helps create policies and drive committees and community projects.
{% endstep %}

{% step %}
### Trustee


{% endstep %}
{% endstepper %}
